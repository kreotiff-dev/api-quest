Базовый URL

Все эндпоинты доступны по базовому URL: http://api.example.com

Аутентификация

Для доступа к некоторым эндпоинтам требуется JWT токен. Для получения токена используйте эндпоинт /api/auth/login.

Токен необходимо передавать в заголовке Authorization в формате Bearer <token>.

Ресурсы

Пользователи

GET/api/users

Возвращает список всех пользователей.

Параметры запроса

Имя Тип Описание Обязательный
limit integer Максимальное количество записей (по умолчанию 10) Нет
page integer Номер страницы (по умолчанию 1) Нет

Заголовки ответа

Имя Описание
Content-Type application/json
X-Total-Count Количество записей

Ответы

200 OK 
{
  "users": [
    {
      "id": 1,
      "name": "John Doe",
      "email": "john@example.com",
      "role": "admin"
    },
    {
      "id": 2,
      "name": "Jane Smith",
      "email": "jane@example.com",
      "role": "user"
    }
  ]
}

Имя Описание

Ответы

POST/api/users

Создает нового пользователя.

Заголовки запроса

Имя Описание Обязательный
Content-Type application/json Да

Тело запроса

{
  "name": "New User",
  "email": "newuser@example.com",
  "password": "password123",
  "role": "user"
}

Параметры запроса

Имя Тип Описание Обязательный
name string Имя пользователя Да
email string Email пользователя (должен быть уникальным) Да
password string Пароль (минимум 6 символов) Да
role string Роль пользователя (по умолчанию "user") Нет

Заголовки ответа

Имя Описание
Content-Type application/json
Location /api/users/{id}

Ответы

201 Created 
{
  "id": 3,
  "name": "New User",
  "email": "newuser@example.com",
  "role": "user"
}
400 Bad Request 
{
  "error": "Неверный формат данных",
  "details": ["Email должен быть валидным", "Пароль должен содержать минимум 6 символов"]
}

Коды статусов

Код Описание
200 OK Запрос выполнен успешно
201 Created Ресурс успешно создан
204 No Content Запрос выполнен успешно, в ответе нет содержимого
400 Bad Request Ошибка в запросе
401 Unauthorized Требуется аутентификация
403 Forbidden Нет доступа к ресурсу
404 Not Found Ресурс не найден
405 Method Not Allowed Метод не разрешен для данного ресурса
500 Internal Server Error Внутренняя ошибка сервера
