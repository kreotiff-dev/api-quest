Документация по API-источникам

API Практикум предлагает несколько источников API для обучения и практики. Каждый источник имеет свои особенности, преимущества и ограничения.

Доступные источники API

Симулятор API Всегда доступен
Локальные моки для обучения без внешних зависимостей. Симулятор предоставляет предсказуемые ответы и не требует подключения к интернету.

Особенности:

  • Работает полностью в браузере, не требует подключения к интернету
  • Предсказуемые ответы на запросы
  • Поддержка всех HTTP методов и форматов данных
  • Имитация задержек сети и ошибок
  • Идеально подходит для начинающих
GET /api/users HTTP/1.1 Host: api-quest-simulator Accept: application/json // Ответ { "status": 200, "body": [ { "id": 1, "name": "Иван Петров", "email": "ivan@example.com", "role": "admin" }, { "id": 2, "name": "Мария Сидорова", "email": "maria@example.com", "role": "user" } ] }
Публичные API Лимиты запросов
Набор бесплатных публичных API для практики с реальными данными. Позволяет получить опыт работы с внешними сервисами.

Особенности:

  • Реальные данные и ответы от публичных сервисов
  • Работа с внешними ограничениями и особенностями API
  • Ознакомление с различными форматами данных и конвенциями именования
  • Ограничения на количество запросов (обычно 60-100 запросов в час)
  • Требуется подключение к интернету
GET /posts HTTP/1.1 Host: jsonplaceholder.typicode.com Accept: application/json // Ответ [ { "userId": 1, "id": 1, "title": "sunt aut facere repellat provident...", "body": "quia et suscipit\nsuscipit recusandae..." }, { "userId": 1, "id": 2, "title": "qui est esse", "body": "est rerum tempore vitae\nsequi sint..." } ]
Учебный API Требуется авторизация
Собственный API платформы с расширенными возможностями для продвинутого обучения. Включает авторизацию, сложные сценарии и специализированные эндпоинты.

Особенности:

  • Полный спектр HTTP методов и возможностей REST API
  • Аутентификация и авторизация (API ключи, токены, OAuth)
  • Валидация данных и обработка ошибок
  • Пагинация, фильтрация и сортировка
  • Сложные сценарии использования API, приближенные к реальным
  • Требуется регистрация и авторизация
POST /api/protected/users HTTP/1.1 Host: api-quest.example.com Content-Type: application/json X-API-Quest-Auth: your_auth_token_123 { "name": "Новый Пользователь", "email": "new@example.com", "role": "user" } // Ответ { "id": 4, "name": "Новый Пользователь", "email": "new@example.com", "role": "user", "createdAt": "2023-07-15T10:30:00Z" }

Использование селектора источников API

Вы можете выбрать источник API для каждого задания с помощью селектора в верхней части экрана. Некоторые задания могут требовать использования конкретного источника API.

Советы по выбору источника:

  • Для начинающих: Используйте Симулятор API, чтобы освоить основы без внешних зависимостей
  • Для изучения внешних API: Переключитесь на Публичные API, чтобы работать с реальными данными
  • Для продвинутых сценариев: Используйте Учебный API для изучения авторизации и сложных сценариев
  • При проблемах с подключением: Вернитесь к Симулятору API, если внешние источники недоступны

Отказоустойчивость и переключение источников

API Практикум автоматически проверяет доступность выбранных источников API и предлагает альтернативы в случае недоступности.

Механизмы отказоустойчивости:

  • Автоматическая проверка доступности источников API при запуске и с интервалом в 60 секунд
  • Автоматическое переключение на доступные источники по приоритету, если текущий источник недоступен
  • Возможность ручного выбора источника API в любой момент
  • Сохранение выбора источника между сессиями
  • Предупреждения при недоступности требуемого для задания источника API

Решение проблем

Если у вас возникли проблемы с источниками API:

  • Проверьте подключение к интернету для Публичных и Учебных API
  • Убедитесь, что вы авторизованы для доступа к Учебному API
  • Временно переключитесь на Симулятор API, если внешние источники недоступны
  • Обновите страницу, чтобы повторно проверить доступность источников
  • Если проблемы сохраняются, обратитесь за помощью к AI-ассистенту
